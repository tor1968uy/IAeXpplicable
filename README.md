# IAeXpplicable

Proyecto para IA Explicable

**Desafío IA Transparente: Explicabilidad en la Detección de Fake News**

Todos los días, miles de publicaciones se viralizan con información no verificada. Algunas son verdaderas, pero muchas contienen falsedades que influyen en la opinión pública y pueden generar consecuencias negativas.

La misión en este desafío es desarrollar un modelo de IA que no solo detecte fake news, sino que además explique por qué una noticia es considerada falsa. El modelo no solo debe ser preciso, sino también comprensible y justo.
